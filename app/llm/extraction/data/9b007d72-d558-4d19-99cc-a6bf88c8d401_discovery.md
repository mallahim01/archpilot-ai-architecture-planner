- Custom build with Next.js (web) + NestJS API + Postgres + Prisma + managed auth (Clerk/Auth0) + Stripe Billing
- Fallback option includes WordPress + LearnDash/LifterLMS + Stripe add-ons with a custom service for payouts/reporting
- User approved the primary custom stack choice.
- Roles for initial launch: Admin, Student, Instructor.
- Subscriptions initially unlock all courses.
- Individual course purchases will be considered a later/optional phase.
- Subscription plans: Monthly and Annual.
- Instructor payouts based on enrollments, either as a fixed amount per enrollment or a revenue-share per enrollment.
- Payouts calculated monthly with an admin-approved payout run.
